# 주간 루틴 매니저 📅

요일별 생활계획표와 알림 기능이 있는 PWA(Progressive Web App) 기반 루틴 관리 애플리케이션입니다.

## ✨ 주요 기능

### 📋 스케줄 관리
- **요일별 스케줄 등록**: 월요일부터 일요일까지 각 요일별로 다른 루틴 설정 가능
- **시간별 활동 관리**: 정확한 시간과 함께 활동명, 상세 설명 입력
- **완료 상태 추적**: 각 스케줄의 완료/미완료 상태 관리
- **실시간 수정/삭제**: 언제든지 스케줄 내용 수정 및 삭제 가능

### 🔔 스마트 알림 시스템
- **브라우저 알림**: Web Notification API를 활용한 시간 기반 알림
- **5분 전 사전 알림**: 놓치지 않도록 미리 알림 제공
- **알림 권한 관리**: 사용자 친화적인 알림 설정 인터페이스
- **스누즈 기능**: 알림을 일시적으로 미루는 기능

### 📊 통계 및 분석
- **실행률 추적**: 전체 및 요일별 스케줄 달성률 확인
- **주간 현황**: 이번 주 실행 현황을 한눈에 파악
- **진행률 시각화**: 직관적인 차트로 성과 확인

### 📱 PWA 기능
- **앱처럼 설치**: 브라우저에서 설치하여 네이티브 앱처럼 사용
- **오프라인 지원**: 인터넷 없어도 기본 기능 사용 가능
- **반응형 디자인**: PC, 태블릿, 모바일 모든 기기에서 최적화
- **빠른 로딩**: 서비스 워커를 통한 캐시 최적화

## 🚀 시작하기

### 1. 파일 다운로드
프로젝트 파일들을 로컬에 다운로드하거나 클론합니다.

### 2. 로컬 서버 실행
PWA 기능을 제대로 사용하려면 HTTPS 또는 localhost에서 실행해야 합니다.

```bash
# Python 3를 사용하는 경우
cd weekly-routine-manager
python -m http.server 8000

# Node.js serve 패키지를 사용하는 경우
npx serve .

# VS Code Live Server 확장 프로그램 사용 권장
```

### 3. 브라우저에서 접속
```
http://localhost:8000
```

### 4. PWA 설치 (선택사항)
- 브라우저 주소창 옆의 설치 아이콘 클릭
- 또는 브라우저 메뉴에서 "앱 설치" 선택

## 💡 사용 방법

### 첫 스케줄 등록하기
1. 상단의 요일 탭에서 원하는 요일 선택
2. "스케줄 추가" 버튼 클릭
3. 시간, 활동명, 설명(선택) 입력
4. 알림 받기 체크 (원하는 경우)
5. "저장" 버튼 클릭

### 알림 설정하기
1. 우상단의 "알림 설정" 버튼 클릭
2. 브라우저 알림 권한 허용
3. 스케줄 등록 시 "알림 받기" 체크박스 선택

### 스케줄 관리하기
- **완료 표시**: 녹색 체크 버튼 클릭
- **수정**: 파란색 편집 버튼 클릭
- **삭제**: 빨간색 휴지통 버튼 클릭

### 키보드 단축키
- `Ctrl/Cmd + N`: 새 스케줄 추가
- `Esc`: 모달 창 닫기
- `1-7`: 요일 전환 (월요일~일요일)

## 🛠️ 기술 스택

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: CSS Grid, Flexbox, CSS Variables 활용
- **Vanilla JavaScript**: 프레임워크 없는 순수 자바스크립트
- **Web APIs**: Notification, Service Worker, Cache API

### PWA 기술
- **Service Worker**: 캐싱 및 오프라인 지원
- **Web App Manifest**: 앱 설치 및 메타데이터
- **Cache API**: 효율적인 리소스 캐싱

### 데이터 저장
- **LocalStorage**: 클라이언트 사이드 데이터 저장
- **JSON**: 구조화된 데이터 형식

## 🔧 커스터마이징

### 색상 변경
`css/style.css`의 CSS 변수를 수정하여 앱의 색상 테마를 변경할 수 있습니다:

```css
:root {
    --primary-color: #4F46E5;  /* 메인 색상 */
    --secondary-color: #10B981; /* 보조 색상 */
    --background-color: #F8FAFC; /* 배경색 */
    /* ... */
}
```

### 알림 시간 변경
`js/notification.js`에서 기본 알림 시간(현재 5분 전)을 변경할 수 있습니다.

### 데이터 백업/복원
개발자 도구 콘솔에서 다음 함수들을 사용할 수 있습니다:
- `storage.exportData()`: 데이터 내보내기
- `storage.importData(file)`: 데이터 가져오기
- `storage.clearAllData()`: 모든 데이터 삭제
